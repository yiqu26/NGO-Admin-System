## 🚨 常見問題快速解決

### 問題 1: API 啟動失敗

```
原因：SQL Server 未啟動
解決：
1. 開啟「服務」(services.msc)
2. 找到 SQL Server (MSSQLSERVER)
3. 右鍵 → 啟動
```

### 問題 2: 前端無法連接 API

```
原因：後端未先啟動
解決：確認步驟 1 的後端已經啟動並看到 "Now listening" 訊息
```

### 問題 3: Port 被占用

```batch
# 查看占用 Port 的程序
netstat -ano | findstr :5264
netstat -ano | findstr :5173

# 關閉程序（替換 <PID> 為實際的進程 ID）
taskkill /PID <PID> /F
```

更多問題解決：參閱 [04-常見問題.md](04-常見問題.md)
