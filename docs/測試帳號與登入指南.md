## 📊 查看現有測試帳號

### 方法 1：使用 SSMS 查詢

在 SQL Server Management Studio 執行：

```sql
-- 查看所有工作人員帳號
SELECT
    WorkerId,
    Email,
    Name,
    Role,
    IsActive,
    CreatedAt
FROM Workers
WHERE IsActive = 1
ORDER BY CreatedAt DESC;
```

### 方法 2：查看帳號角色說明

```sql
-- 查看各角色的帳號數量
SELECT
    Role,
    COUNT(*) AS AccountCount,
    STRING_AGG(Email, ', ') AS Emails
FROM Workers
WHERE IsActive = 1
GROUP BY Role;
```

**角色說明**：
- `admin` - 管理員（完整系統權限）
- `supervisor` - 督導（管理和審核權限）
- `staff` - 一般員工（基本操作權限）

## ➕ 創建測試帳號

如果資料庫中沒有測試帳號，可以使用以下 SQL 腳本創建：

### 創建管理員測試帳號

```sql
-- 創建管理員帳號
-- Email: admin@ngo.org
-- 密碼: Admin123!
INSERT INTO Workers (Email, Password, Name, Role, IsActive, CreatedAt, UpdatedAt)
VALUES (
    'admin@ngo.org',
    '$argon2id$v=19$m=19456,t=2,p=1$YOUR_HASH_HERE', -- 需要使用 Argon2 加密
    '系統管理員',
    'admin',
    1,
    GETDATE(),
    GETDATE()
);
```

### ⚠️ 密碼加密問題

資料庫中的密碼使用 **Argon2** 加密，不能直接存明文。有兩個方法：

#### 方法 1：使用 API 註冊端點（推薦）

如果 WebAPI 有提供註冊端點，可以使用 Postman 或瀏覽器測試：

```
POST http://localhost:5264/api/Worker/register
Content-Type: application/json

{
  "email": "test@ngo.org",
  "password": "Test123!",
  "name": "測試員工",
  "role": "staff"
}
```

#### 方法 2：查看資料庫現有帳號

如果資料庫已經有帳號，可以直接使用：

```sql
-- 查看現有帳號（含密碼hash，僅供參考）
SELECT TOP 5
    WorkerId,
    Email,
    Name,
    Role,
    LEFT(Password, 50) + '...' AS PasswordHash
FROM Workers
WHERE IsActive = 1;
```

**然後詢問組員這些帳號的原始密碼**

#### 方法 3：臨時修改為簡單密碼驗證（僅測試用）

可以在 WebAPI 中暫時新增一個測試端點來創建帳號，或者詢問組員是否有種子數據（Seed Data）腳本。

## 🚨 常見問題排除

### Q1: 登入時顯示"找不到對應的工作人員帳號"
**原因**：資料庫中沒有該 Email 的帳號

**解決方式**：
1. 檢查資料庫中是否有測試帳號
2. 使用 SQL 查詢確認 Email 是否正確
3. 創建新的測試帳號

### Q2: 密碼錯誤
**原因**：輸入的密碼與資料庫中加密的密碼不符

**解決方式**：
1. 確認密碼是否正確（詢問組員）
2. 檢查 WebAPI 的密碼驗證邏輯
3. 查看 WebAPI 日誌確認錯誤訊息

### Q3: API 連線失敗
**檢查清單**：
1. WebAPI 是否正在運行？（http://localhost:5264）
2. React 的 `.env.development` 中 API 位址是否正確？
3. 瀏覽器 Console 是否有 CORS 錯誤？

### Q4: 資料庫連線失敗
**檢查清單**：
1. SQL Server 服務是否啟動？
2. `appsettings.json` 中的連線字串是否正確？
3. 資料庫名稱是否為 `NGOPlatformDb`？

## 📝 找出現有帳號的完整 SQL

```sql
-- 完整查詢：查看所有可用帳號及其詳細資訊
SELECT
    WorkerId AS [帳號ID],
    Email AS [登入Email],
    Name AS [姓名],
    CASE
        WHEN Role = 'admin' THEN '管理員'
        WHEN Role = 'supervisor' THEN '督導'
        WHEN Role = 'staff' THEN '一般員工'
        ELSE Role
    END AS [角色],
    CASE
        WHEN IsActive = 1 THEN '啟用'
        ELSE '停用'
    END AS [狀態],
    CreatedAt AS [建立時間]
FROM Workers
WHERE IsActive = 1
ORDER BY Role, CreatedAt DESC;

-- 查看帳號總數
SELECT
    COUNT(*) AS [啟用帳號總數],
    SUM(CASE WHEN Role = 'admin' THEN 1 ELSE 0 END) AS [管理員],
    SUM(CASE WHEN Role = 'supervisor' THEN 1 ELSE 0 END) AS [督導],
    SUM(CASE WHEN Role = 'staff' THEN 1 ELSE 0 END) AS [員工]
FROM Workers
WHERE IsActive = 1;
```
