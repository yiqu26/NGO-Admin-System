## 📋 系統簡介

這是一個完整的 NGO 管理系統，包含：
- **前端**：React + TypeScript + Material-UI
- **後端**：ASP.NET Core WebAPI

## 🚀 快速啟動（展示模式）

### 步驟 1：執行啟動腳本

雙擊執行：
```
C:\Users\lanli\source\repos\啟動展示系統.bat
```

這個腳本會自動啟動：
1. 後端 WebAPI (localhost:5264)
2. 前端 React (localhost:5173)
3. ngrok 後端 tunnel
4. ngrok 前端 tunnel

### 步驟 2：取得公開網址

啟動完成後，瀏覽器會自動開啟 ngrok 管理介面：
```
http://localhost:4040
```

你會看到兩個 tunnel：
- **前端網址**：`https://xxxx.ngrok-free.app` (port 5173)
- **後端網址**：`https://yyyy.ngrok-free.app` (port 5264)

### 步驟 3：測試系統

1. 複製**前端網址**
2. 用瀏覽器開啟
3. 使用測試帳號登入：
   ```
   Email: admin@ngo.org
   密碼: Admin123!
   ```

## 📞 問題排除

### Q1: 啟動腳本失敗
**解決方式**：
1. 確認 SQL Server 正在運行
2. 確認 ngrok 已安裝並設定 authtoken
3. 確認 port 5264 和 5173 沒有被占用

### Q2: ngrok 顯示警告頁面
**原因**：ngrok 免費版的安全提示

**解決方式**：
- 點擊「Visit Site」按鈕繼續
- 或告訴面試官這是 ngrok 的正常提示

### Q3: 前端無法連接後端
**檢查**：
1. 兩個 ngrok tunnel 都在運行嗎？
2. 瀏覽器 Console 有錯誤嗎？
3. 後端 API 正常運行嗎？

**測試**：
```
# 測試後端
curl https://yyyy.ngrok-free.app

# 應該回應: NGO API 運作正常
```

### Q4: 登入失敗
**檢查**：
1. 確認使用正確的測試帳號
2. 確認資料庫有測試帳號資料
3. 檢查後端 API 是否正常
